# Файлы для импорта товаров

Эта папка содержит примеры файлов для импорта товаров в систему.

## Поддерживаемые форматы

- CSV (`.csv`)
- Excel (`.xlsx`, `.xls`)

## Формат данных

Файлы должны содержать следующие колонки:

### Обязательные поля:
- `store_name` - название магазина (строка)
- `name` - название товара (строка)
- `price` - цена товара (число, можно с запятой или точкой)

### Опциональные поля:
- `description` - описание товара (строка)
- `sku` - артикул товара (строка)
- `stock_quantity` - количество на складе (целое число, по умолчанию 0)
- `is_available` - доступность товара (true/false, 1/0, да/нет, по умолчанию true)

## Пример использования

### Импорт из CSV:
```bash
python manage.py import_products import_files/example_products.csv
```

### Импорт из Excel:
```bash
python manage.py import_products import_files/products.xlsx
```

### Импорт с указанием листа Excel:
```bash
python manage.py import_products import_files/products.xlsx --sheet "Товары"
```

### Проверка без сохранения (dry-run):
```bash
python manage.py import_products import_files/example_products.csv --dry-run
```

## Поведение импорта

- Если магазин с указанным названием не существует, он будет создан автоматически
- Если товар с таким SKU уже существует в магазине, он будет обновлен
- Если товар с таким SKU не найден, но есть товар с таким же названием в магазине, он будет обновлен
- Если товар не найден, будет создан новый
